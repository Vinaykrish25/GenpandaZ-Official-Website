# GenpandaZ - Digital Agency

GenpandaZ is a data-driven digital agency specializing in web development, digital marketing, video editing, graphic design, and automation.

## Features

- **Web Development**: Modern, high-performance web applications built with React and Vite.
- **Digital Marketing**: Data-driven strategies to grow your brand.
- **Video Editing & Graphic Design**: Professional visual content.
- **Automation**: Streamlining workflows with smart solutions.

## Technologies

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Development

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Setup

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

The project can be deployed to any static site hosting service (Vercel, Netlify, GitHub Pages, etc.) by building the project and uploading the `dist` directory.
